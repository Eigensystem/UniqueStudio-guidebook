## 全球共享的分布式计算机真的可行吗？自由的全球银行如何记账？金融系统的高效运转由谁来实现？不公平的定价和费率如何被套利交易纠正？虚拟记录又如何与现实世界无缝对接？

这些问题的答案，由你来定义。

在联创，创始人、团队成员和学长们不约而同地提出了一个共识——我们应该探索更多新兴前沿领域。于是，Project C 应运而生，承载着我们对未来的最新想象。

## 关于 Project C

Project C 立足计算机科学与工程，专注于区块链技术与金融科技，承接 Lab 组的技术积累，探索海量数据与计算的无限可能。

## 我们要做的

1. 全栈技术：数据获取、分析、可视化、监控（Web 组、AI 组交叉方向）
2. 区块链智能合约：开发与实现（Lab 组交叉方向）
3. 分布式金融系统与产品研究（PM 组交叉方向）
4. 区块链 Paper Read Group
5. ……

## 我们希望你

1. 具备一技之长：Project C 是一个交叉应用方向，我们希望你在联创涉及的某个领域有深入的经验。
2. 对社会经济运作充满好奇：愿意包容新生事物，并具备出色的信息检索与学习能力。
3. 硬核计算机科学家：能从技术和数据的角度解构金融活动。
4. 冷静理性：能"冷眼旁观"投机行为，以数据和逻辑洞察市场。

## 招新要求

1. 对区块链技术有浓厚兴趣，对**区块链技术有一定基础**。
2. 达到联创团队其他组的招新标准。
3. 有较强的学习能力。

我们将在正常招新流程之外对你进行单独一轮的面试，请在报名时勾选“**Project C**”。

## 我们提供的

1. 接触业界前沿，对接行业大佬交流机会。
2. 区块链与金融行业的专业知识，助你构建深厚的认知体系。
3. 一群志同道合的优秀伙伴，共同进步，相互促进。

Project C 不是一个独立组别，而是团队内部的Part Time研究方向。其他组别的同学将在这里将获得相关的但不同的"连携"版新人任务。

Project C 只是起点，未来将由你定义！ 🚀

# 区块链与加密货币入门指南

## 基础篇：区块链与加密货币基础

### 学习目标

- 理解区块链基本原理
- 掌握加密货币基础知识
- 了解DeFi生态系统
- 建立自己的第一个加密钱包

### 核心学习资源

### 1. 区块链技术基础

- **视频教程**: [3Blue1Brown的"比特币和区块链是如何工作的"](https://www.youtube.com/watch?v=bBC-nXj3Ng4) - 通过可视化方式解释区块链的基本原理
- **在线课程**: [Coursera - 区块链基础知识](https://www.coursera.org/learn/blockchain-basics) & [PKU学生区块链中心WEB3新人公开课](https://space.bilibili.com/586660955/lists/894352)
- **书籍**: 《精通比特币》(Mastering Bitcoin) - Andreas Antonopoulos著
- **指南**: [区块链技术指南](https://yeasy.gitbook.io/blockchain_guide/) - 完整的中文区块链入门资料

### 2. 加密货币基础

- **官方文档**: [比特币白皮书](https://bitcoin.org/bitcoin.pdf) - 了解比特币的设计理念
- **视频：**[Whiteboard Crypto](https://www.youtube.com/c/WhiteboardCrypto) - 简化的加密货币教学
- **学习平台**: [Binance Academy](https://academy.binance.com/zh/start-here) - 币安学院提供全面的加密货币入门知识

### 3. DeFi基础

- **概念理解**: [什么是DeFi?](https://ethereum.org/zh/defi/) - 以太坊官方对DeFi的解释
- **教程系列**: [DeFi Prime](https://defiprime.com/) - DeFi应用目录与学习资源
- **视频课程**: [Finematics](https://www.youtube.com/c/Finematics) - 专注DeFi教学的YouTube频道

### 4. 实践项目

1. **创建和管理加密钱包**
    - 创建MetaMask钱包
    - 学习私钥管理和安全
    - 在测试网络获取测试代币
2. **体验基本的区块链交互**
    - 在测试网络上发送交易
    - 使用区块浏览器如[Etherscan](https://etherscan.io/)查看交易
    - 参与简单的代币交换(使用Uniswap测试网)
3. **学习工具**:
    - [MetaMask](https://metamask.io/) - 以太坊钱包
    - [Etherscan](https://etherscan.io/) - 以太坊区块浏览器
    - [Remix IDE](https://remix.ethereum.org/) - 在线智能合约开发环境

### 推荐学习路径

1. 首先通过3Blue1Brown的视频建立区块链的直观理解
2. 阅读《精通比特币》的前3-5章，掌握基本概念
3. 完成Binance Academy的"区块链基础"和"加密货币基础"课程
4. 创建自己的MetaMask钱包，尝试在测试网络上进行基本操作
5. 探索基本的DeFi概念，尝试使用测试网络上的DEX

## 进阶篇：智能合约与DeFi深度学习

### 学习目标

- 掌握智能合约开发基础
- 理解主流DeFi协议原理
- 学习链上数据分析方法
- 开发简单的DeFi应用

### 核心学习资源

### 1. 智能合约开发

- **官方教程**: [Solidity文档](https://docs.soliditylang.org/) - 智能合约语言官方指南
- **实践指南**: [CryptoZombies](https://cryptozombies.io/) - 互动式智能合约开发课程
- **GitHub资源**: [智能合约应用指南](https://github.com/33357/smartcontract-apps) - 包含智能合约入门、实战和事件分析
- **视频课程**: [Chainlink工程师训练营](https://chain.link/bootcamp) - 全面的区块链开发课程

### 2. DeFi协议解析

- **Uniswap**: [Uniswap文档](https://docs.uniswap.org/) - 学习AMM原理
- **Aave**: [Aave开发者文档](https://docs.aave.com/developers/) - 了解借贷协议
- 链上套利: [Uniswap套利分析](https://github.com/antaintan/uniswap-arbitrage-analysis) - 学习DEX套利机制

### 3. 区块链数据分析

- **学习资源**: [精通链上数据分析](https://github.com/SixdegreeLab/MasteringChainAnalytics) - 从入门到高级的链上数据分析教程
- **分析平台**: [Dune Analytics](https://dune.com/) - 区块链数据可视化平台
- **数据API**: [The Graph](https://thegraph.com/) - 区块链数据索引协议

### 4. 多链开发

- **Solana**: [Solana开发文档](https://solana.com/zh/docs) - Solana区块链开发指南
- **跨链技术**: [LayerZero文档](https://layerzero.network/developers) - 学习跨链应用开发

### 5. 实践项目

1. **开发智能合约**
    - 使用Remix IDE开发ERC-20代币
    - 创建简单的DeFi应用(如质押、Farm)
    - 学习合约安全最佳实践
2. **DeFi协议分析**
    - 分析Uniswap的智能合约代码
    - 理解借贷协议的风险模型
    - 研究流动性挖矿机制
3. **链上数据分析**
    - 使用Dune Analytics创建DeFi数据看板
    - 跟踪协议TVL、交易量等关键指标
    - 实现简单的链上行为分析
4. **工具与资源**:
    - [Hardhat](https://hardhat.org/) - 以太坊开发环境
    - [OpenZeppelin](https://openzeppelin.com/) - 安全智能合约库
    - [Tenderly](https://tenderly.co/) - 智能合约监控和调试工具

### 推荐学习路径

1. 完成CryptoZombies课程，掌握Solidity基础
2. 阅读并实践智能合约应用指南
3. 深入研究一个DeFi协议的代码(推荐Uniswap V2)
4. 学习精通链上数据分析课程，使用Dune构建分析看板
5. 尝试创建自己的DeFi应用(如简单的借贷协议或流动性池)

## 高级篇：量化交易与高频策略

### 学习目标

- 构建加密货币量化交易系统
- 开发高频交易策略
- 理解MEV和链上套利机会
- 掌握交易机器人开发

### 核心学习资源

### 1. 量化交易基础

- **课程**: [QuantConnect课程](https://www.quantconnect.com/tutorials/crypto)
- **书籍**: 《量化交易:如何建立自己的算法交易业务》- Ernest Chan著
- **开源框架**: [Freqtrade](https://www.freqtrade.io/) - 开源加密货币交易机器人

### 2. 高频交易与MEV

- **研究资源**: [Flashbots研究](https://docs.flashbots.net/) - MEV研究与工具
- **MEV工具**: [MEV-Boost](https://boost.flashbots.net/) - 以太坊MEV解决方案
- **视频讲解**: [MEV in DeFi](https://www.youtube.com/watch?v=HNDkNXD0m20) - 详解DeFi中的MEV现象

### 3. 交易机器人开发

- **开源项目**: [币安自动下单机器人](https://github.com/duckdegen/apebot) - 高效的链上交易机器人示例
- **套利系统**: [Hummingbot](https://hummingbot.io/) - 开源加密货币做市和套利软件
- **策略开发**: [Uniswap V3策略](https://y1cunhui.github.io/uniswapV3-book-zh-cn/docs/introduction/uniswap-v3/) - Uniswap V3策略设计与分析

### 4. 高级系统架构

- **基础设施**: [Infura](https://www.infura.io/) - 区块链节点即服务
- **开发框架**: [Brownie](https://eth-brownie.readthedocs.io/) - Python智能合约开发框架
- **性能优化**: [Web3.py优化](https://web3py.readthedocs.io/en/stable/middleware.html) - Web3交互性能优化

### 5. 实践项目

1. **构建量化交易系统**
    - 开发基于技术分析的交易策略
    - 实现风险管理模块
    - 回测与优化策略参数
2. **链上套利机器人**
    - 开发DEX间套利机器人
    - 实现闪电贷套利策略
    - 优化Gas使用和交易时机
3. **MEV策略研究**
    - 分析三明治攻击机制
    - 开发抢先交易(frontrunning)检测工具
    - 研究区块打包优化
4. **高频交易基础设施**
    - 构建低延迟区块链数据提取系统
    - 开发多链监控架构
    - 实现交易策略的自动化调整
5. **工具与资源**:
    - [TradingView](https://www.tradingview.com/) - 技术分析平台
    - [Chainlink Price Feeds](https://chain.link/data-feeds) - 去中心化价格预言机
    - [Foundry](https://github.com/foundry-rs/foundry) - 高性能智能合约开发工具

### 推荐学习路径

1. 学习量化交易基础知识，掌握基本策略
2. 研究并实践基本的链上套利策略
3. 深入了解MEV原理，分析现有MEV策略
4. 开发自己的交易机器人，从简单策略开始
5. 学习高性能系统设计，优化交易延迟和执行效率
6. 实现复杂的多协议、多链套利策略
