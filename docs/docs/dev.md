# 二. 拆分原方程（联创团队介绍）

联创团队（Unique Studio）创建于2000年6月。队名“联创”寓意联众人之志，创非凡之事。我们秉承不断创新和锐意进取的核心精神，立足互联网与信息技术，旨在建立一个精英学生的IT技术平台。作为一个以兴趣为主导、强调主动实践和自主创新的学生团队，自由、自律、自驱是联创的团队精神。

团队致力于为有潜力、肯努力并富有激情的学生构建一流的学习环境，让兴趣变成能力，引导他们快速成长。

目前团队设有以下七个小组，为拥有不同专业能力的学生提供个性化的资源：

- AI 组
- Mobile 组
- Design 组
- Game 组
- Lab 组
- PM 组
- Web 组

（摘自24年春招）

# 三. 选择你的公式（各组介绍及招新要求）

## Mobile组 / Lab组

### Lab组介绍
Lab组是Geek们的聚集地。在这里，我们深入计算机系统底层原理、编写高性能代码、处理海量数据。高性能计算、服务器运维、分布式系统、操作系统、编程语言…我们以计算机系统为根基，致力于探索各种计算机技术。

### Lab组要求
- 能熟练使用 Linux 作为开发环境
- 能熟练使用 C 语言
- 对计算机网络、操作系统、分布式、高性能计算、数据库、体系结构等领域中的至少一项有较为深入的了解或者深度探索的兴趣

#### 加分项
- 有两种及以上常见编程语言的学习经历
- （在满足基本要求的基础上）参加过算法竞赛或者其他计算机科学领域的竞赛
- 阅读过《CSAPP》的部分内容
- 能够使用 git，在 Github 等平台上传有自己的完整项目

---

## PM组

### PM组介绍
PM组聚集了一群有趣的产品经理。我们是用户与开发技术人员之间沟通的桥梁，以设计出能够满足用户需求的产品为目标，在产品的开发过程中致力于对产品功能等方面的诠释，将天马行空的想法落地实施。 PM组主要分为产品和运营两个方向。产品方向的工作内容包括学习经典产品结构模式、探索前沿技术的产品应用、组织开发程序落地有趣的项目构想等；运营方向的工作内容则包括策划联创团队日常活动、利用团队资源举办大型活动、运营自媒体平台、与业内前辈建立联系并交流，与其他高校团队联络合作等。

### PM组要求
- 有快速学习的能力，拥有热情与好奇心
- 有较强的沟通能力、逻辑思维能力和写作能力，对产品或运营有基本认知
- 擅于思考，注重细节，能够洞察事物本质，时常能领略到生活中的“不合理”
- 互联网使用程度高，深入体验过大量互联网产品，并对其有自己独到的见解和分析
- 对互联网有热情，对生活有思考，渴望拥有一个不被束缚、不被规训的大学生活

#### 加分项
- 有组织策划活动经验
- 有与产品相关的项目经历
- 具备相关软件基础或有公众号、自媒体运营经验

---

## Design组

### Design组介绍
我们并非美工，而是设计师。在这里，你可以任意发挥艺术天赋与设计思维，并借助技术和互联网自由地设计。

Design 组分为应用交互和游戏设计两个方向：
- **应用交互方向** 主要从事各平台应用和网页的视觉及交互设计，也会涉及平面设计和动效/视频制作。
- **游戏设计方向** 主要在交互体验 GUI、原画等领域进行设计创作。

### Design组要求
1. 拥有基础设计知识，熟悉 Figma、XD、Sketch、PS、AI、AE 等设计软件操作
2. 对 B、C 端交互设计师的工作内容有一定熟悉度，能够胜任团队事务（应用交互方向）
3. 对角色/场景原画、交互体验设计、GUI、角色/场景 3D、动画与特效等领域有一定了解（游戏设计方向，只需了解其中一个领域即可）

**注意：**  
报名时请附带个人作品集（推荐上传 PDF）

## AI组

### AI组介绍
AI组基于人工智能技术，用智慧创造“智慧”。在这里，我们紧跟学术界最前沿，从底层原理到上层应用，专注于学术型、技术型的研究，同时注重实践产出。从 Kaggle 到各种 AI 比赛，让所学变为所用。主要方向有机器学习、强化学习、计算机视觉、自然语言处理、语音识别与合成。

### AI组要求
- 有良好的数学基础
- 掌握 Python 编程
- 了解基本的神经网络，并会使用 PyTorch（推荐）或 TensorFlow 等深度学习库
- 基本掌握 Linux 的使用和 ssh 连接服务器
- 对 LLM（大语言模型）有基本了解

---

## Game组

### Game组介绍
我们是一群热爱游戏的创作者，致力于设计好玩的游戏，同时也喜欢探索背后的技术。我们涉足游戏设计、图形学、人工智能、物理和网络等多个领域。引擎是我们创造乐趣的工具，策划则是架构世界的蓝图。

### Game组岗位划分与要求

#### 程序岗位（满足其中一项即可）
- 会使用一种面向对象编程语言，并了解基本算法
- 正在学习或已熟悉 Unity/Unreal 等游戏引擎，有参与过游戏开发更好
- 对游戏 AI、图形学或引擎开发有一定了解和学习经验
- 了解 Shader 编程、渲染管线或视觉效果实现，能够将技术与艺术结合

#### 策划岗位（满足其中一项即可）
- 尝试过游戏数值设计或关卡设计，文笔不错，对科幻、武侠等题材感兴趣，写过故事或参与过影视创作
- 玩过各种类型的游戏，并对游戏设计流程有一定理解
- 参与过完整项目并担任过策划角色
- 有基础绘画能力，对角色和场景设计有自己的想法，愿意参与 UI 设计和视觉风格定义

**注：**  
不要求你成为专家，只要你热爱游戏、愿意学习和合作，就来加入我们吧！

## Web组

### Web组介绍
从编写 Web 页面，到构建高性能的前端甚至跨平台应用，Web 提供了无限可能；从设计高性能分布式后端应用，到持续部署后端代码至集群，我们不设边界。加入我们，你将学到很多。

### Web组方向与要求

#### 前端开发要求
- 对前端有浓厚兴趣，熟悉 HTML、CSS、JavaScript
- 熟悉页面架构和布局，对 Web 标准和标签语义化有理解
- 了解计算机网络基础知识（HTTP, TCP/UDP, IP…）
- **加分项：** 对 React/Vue/Angular 等任意一种前端主流框架有所了解

#### 后端开发要求
- 掌握至少一种后端开发语言，例如 Golang、Python…
- 掌握 Linux 基本操作及相关知识
- 了解计算机网络基础知识（HTTP, TCP/UDP, IP…）
- 了解数据库相关知识及基础使用（MySQL, Redis, PostgreSQL…）
